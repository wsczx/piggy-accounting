# 快速开始

## 安装

### macOS

1. 下载最新版本的 `猪猪记账-macOS.dmg`
2. 打开 DMG 文件，将应用拖到 Applications 文件夹
3. 首次打开时，前往 系统设置 > 隐私与安全性 允许应用运行

### Windows

1. 下载最新版本的 `猪猪记账-Windows.exe`
2. 双击安装程序，按提示完成安装

### Linux

1. 下载最新版本的 `猪猪记账-Linux.tar.gz`
2. 解压并运行：
   ```bash
   tar -xzf 猪猪记账-Linux.tar.gz
   ./猪猪记账
   ```

## 首次使用

1. 打开应用后，系统会自动创建默认账本
2. 您可以在 个人页 > 多账本 中创建更多账本
3. 在 个人页 > 账户 中添加您的账户（现金、银行卡等）
4. 开始记账！点击底部中央的 + 按钮添加记录

## 基本操作

### 添加记账记录

1. 点击底部导航栏的 + 按钮
2. 选择支出或收入
3. 选择分类和账户
4. 输入金额和备注
5. 点击保存

### 查看统计

点击底部导航栏的 统计 图标，查看：
- 月度收支趋势
- 分类占比饼图
- 账户余额分布

### 设置提醒

前往 个人页 > 提醒设置：
- 设置每日记账提醒时间
- 配置预算预警阈值
- 开启待办任务到期提醒

### 数据备份

前往 个人页 > 备份：
- 点击 立即备份 创建手动备份
- 设置自动备份频率
- 从备份恢复数据

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Cmd/Ctrl + N | 新建记账记录 |
| Cmd/Ctrl + 1 | 切换到首页 |
| Cmd/Ctrl + 2 | 切换到统计 |
| Cmd/Ctrl + , | 打开设置 |
