# 功能介绍

猪猪记账提供完整的个人财务管理功能，以下是详细的功能说明。

## 📊 记账功能

### 收支记录
- 快速记录日常收入和支出
- 支持自定义分类（餐饮、交通、购物等）
- 添加备注和标签
- 关联账户（现金、银行卡、支付宝等）
- 支持搜索和筛选

### 周期记账
- 支持四种周期类型：
  - **每日** - 每天自动记账
  - **每周** - 每周指定星期几记账
  - **每月** - 每月指定日期记账
  - **每年** - 每年指定月份和日期记账
- 自动执行到期记账
- 可启用/禁用规则
- 支持关联账户

## 🏦 账户管理

### 多账户支持
- 创建多个账户（现金、银行卡、信用卡、支付宝、微信等）
- 设置初始余额
- 实时计算账户余额
- 标记默认账户

### 转账记录
- 记录账户间转账
- 自动关联收支记录
- 转账历史查询

## 📈 统计分析

### 数据可视化
- **收支趋势图** - 按月展示收入支出变化
- **分类饼图** - 展示各类别占比
- **账户余额分布** - 各账户余额对比

### 时间维度
- 月度统计
- 年度统计
- 自定义时间范围

## 📋 待办任务

### 任务管理
- 创建记账相关待办事项
- 设置到期日期和金额
- 标记完成状态
- 按到期日排序

### 任务提醒
- 到期前自动提醒
- 可设置提前提醒天数

## 🔔 提醒系统

### 提醒类型
- **每日记账提醒** - 定时提醒记账
- **预算预警** - 支出接近预算时提醒
- **待办任务提醒** - 任务到期前提醒

### 通知方式
- 应用内弹窗提醒
- Webhook 通知（支持钉钉、企业微信、Discord 等）

## 🏷️ 标签系统

- 为记录添加多个标签
- 按标签筛选记录
- 标签管理（增删改）

## 📚 多账本

- 创建多个独立账本
- 账本间数据隔离
- 快速切换账本
- 账本导入导出

## 💾 数据管理

### 备份恢复
- 手动备份
- 自动备份（可设置频率）
- 从备份恢复
- 备份文件导出

### 导入导出
- CSV 格式导出
- CSV 格式导入
- 选择导出时间范围

## 🎨 个性化

### 主题
- 浅色模式
- 深色模式
- 跟随系统

### 界面
- 精美的渐变设计
- 流畅的动画效果
- 响应式布局

## 🔒 安全

- 数据本地存储
- 自动备份保护
- 安全的数据恢复机制

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Cmd/Ctrl + N | 新建记账记录 |
| Cmd/Ctrl + 1 | 切换到首页 |
| Cmd/Ctrl + 2 | 切换到统计 |
| Cmd/Ctrl + 3 | 切换到分类 |
| Cmd/Ctrl + 4 | 切换到任务 |
| Cmd/Ctrl + 5 | 切换到个人页 |
| Cmd/Ctrl + , | 打开设置 |
| Cmd/Ctrl + + | 放大界面 |
| Cmd/Ctrl + - | 缩小界面 |
| Cmd/Ctrl + 0 | 重置缩放 |
